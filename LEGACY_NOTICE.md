# Legacy Notice

Este repositório mantém uma versão **legada / protótipo / local** do **2K Command OS / DC 2K CRM**.

Ele foi preservado como referência histórica e visual para o desenvolvimento do ecossistema 2K Studios, especialmente para:

- layout dark do painel;
- sidebar operacional;
- dashboard financeiro;
- cards de indicadores;
- módulos internos como Dashboard, Produções, Financeiro, Clientes, Agenda, Discord, Obsidian e Automações;
- experimentos de integração local com backend Node.js/Express e frontend React/Vite.

## Status

Este projeto **não é o SaaS principal atual**.

O produto principal atual está em outro repositório:

- `therunnas/2k-web-saas`

## Uso correto deste repositório

Use este projeto como:

- referência visual;
- histórico de evolução;
- protótipo local;
- base de consulta para decisões de UI/UX;
- material de comparação com o novo SaaS.

Não use este projeto como fonte final de arquitetura do novo SaaS.

## Observações

Este repositório pode conter:

- dados mockados;
- módulos experimentais;
- integrações locais;
- executores para Windows/macOS;
- referências antigas a Discord, Obsidian e automações;
- estrutura diferente do projeto SaaS atual.

Qualquer migração para o `2k-web-saas` deve ser feita manualmente, com revisão técnica, sem copiar a estrutura inteira deste projeto para o SaaS novo.
